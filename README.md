# ha-brewbubbles
# Brew Bubbles (Home Assistant Integration)

Custom integration for Brew Bubbles https://github.com/lbussy/brew-bubbles

## Install (HACS)
1. HACS → Integrations → “Custom repositories”
2. Add this repo URL as category **Integration**
3. Install, restart Home Assistant
4. Add integration via Settings → Devices & services

## What it exposes
- BPM (bubbles per minute)
- Vessel temperature (if sensor present)
- Ambient temperature (if sensor present)
- Firmware version / update availability (planned)
